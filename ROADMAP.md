# Roadmap

Planned features for future releases:

- Enable passing type arguments to `.toAcceptProps()`, `.toBeCallableWith()`, `.toBeConstructableWith()`
- Explain type differences in error messages for `.toBe()`, `.toBeAssignableFrom()`, `.toBeAssignableTo()`
- Improve watch mode to handle changes in imported files, `tsconfig.json`, `package.json`
